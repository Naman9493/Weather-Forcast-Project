# Weather Forecast Project 🌦️

A simple Weather Forecast web application that allows users to search for weather information for any city.

## Features

* Search weather by city name
* Displays temperature
* Shows humidity
* Displays wind speed
* User-friendly interface
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## Project Structure

```
Weather-Forcast-Project/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
